Utilities which work with this organization

Some C programs and shell scripts make it a lot easier to run a master source (or meta source) pool. While none of these are (strictly) mandatory they are enabling technologies for certain.

Even the rdist> front end, distrib, could be replaced by some in-line shell code (which I will not include here).

These are referenced in the baseline Makefile and Make.host files.
